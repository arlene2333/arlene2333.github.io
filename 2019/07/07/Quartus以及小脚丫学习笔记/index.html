设计

通过做实验来熟悉这个软件以及了解IP核的使用，明白了锁相环是什么东西

如在实现移频的实验中，就要用到​PLL IP核​来利用小脚丫开发板本身的12MHz频率来生成两个时钟频率10MHZ和10KHz
再将这两个频率的信号相与就可以实现移频

设置ALTPLL时的各种参数如下：


inclk0 input设置为12MHz

clkc0设置为10MHz

clkc1设置为0.01MHz

具体的代码如下：

module usePLL (clk,rst_n,clkout);

    input     clk,rst_n;                       
    output    clkout;                          
      wire[1:0] temp1;
      wire[1:0] temp2;

 pll1 m1(.inclk0(clk),
            .c0(temp1),//10M
            .c1(temp2)

            );
    process m2(
    .clk_in1(temp1),
    .clk_in2(temp2),
    .clk_process(clkout)
    );


endmodule


module process(clk_in1,clk_in2,clk_process);

input clk_in1,clk_in2;

output clk_process;



assign clk_process = clk_in1 & clk_in2;

endmodule

然后在​Add File​中选择pof文件进行添加，如图所示：
